Trabalhando com Fontes

Normalmente no código é colocado uma combinação de fontes, pois alguns dispositivos não reconhecem algumas fontes.

Por isso, geralmente é usado a combinação de: Arial, Helvética, Sans-serif. Esse tipo de declaração é chamada de safe combination - declaração segura.

Subtítulo do Exercício

Para saber quais combinações de fontes usar uma dica é pesquisar no site: Css web Safe Fonts - W3Schools.

Existem várias formas de representar um tamanho de fonte (font-size), sendo que podem ser representadas por medidas absolutas e medidas relativas.

As medidas absolutas são representadas por cm (centímetro), mm (mlímetro),in (polegada), px (pixel), pt (ponto) e pc (paica).

As medidas relativas são representadaas por em (tamnaho atual da fonte), ex (altura x da fonte), rem (fonte configurada no body), vw (largura da viewport - largura da tela), vh (altura da viewport), % (porcentagem).

Para medida da fonte é recomendado o uso de px e do em.

Normalmente o tamanho padrão é 16px, sendo que 16px = 1em.

Peso, Estilo e Shorthand Font

O peso de uma fonte é representado por font-weight e existem as seguintes variações: lighter, normal, bold, bolder.

Para personalizar o estilo de uma fonte é usado o FONT-STYLE:ITALIC (itálico), FONT-WEIGHT:BOLDER (negrito), TEXT-DECORATION:UNDERLINE (sublinhado)

A propriedade SHORTHAND é onde permite colocar várias coisas em uma única linha. Ex: font: italic bolder 3em arial,helvetica,sans-serif.
