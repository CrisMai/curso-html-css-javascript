Usando Fontes Externas Baixadas

Além do site do Google Fonts é possível baixar outros formantos de fontes através do site DaFont

Quando for escolher uma fonte deve tomar o cuidado de observar se a mesma contém todos os glifos, ou seja, se possui letras maiúsclas, minúsculas, letras acentuadas.

Para carregar a fonte selecionada depois de salva nos arquivos, é usado um seletor especial FONT-FACE.

Normalmente essas fontes serão salvas com a extensão otf e ttf...podem ser usadas os dois tipos.

Existem alguns formatos que são usados: opentype (otf); truetype (ttf); embedded-opentype; truetype-aat (Apple Advance Typo Graphy); svg.
