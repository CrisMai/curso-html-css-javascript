Principais Formatações

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK.

Para mudar a cor do texto marcado usamos o parâmetro STYLE.

Se quiser várias palavras com um texto marcado: no cabeçalho HEAD crie a tag STYLE, em seguida crie um seletor {} para MARK e selecione BACKGROUND-COLOR: e a cor que desejar.

Texto Grande e Pequeno

Estamos criando um texto grande usando a tag BIG e um texto pequeno usando a tag SMALL.

Texto Excluído

Podemos marcar um texto como excluído usando a tag DEL, esse texto pode ser lido, mas não deve ser considerado.

Texto Inserido

Podemos marcar um texto como inserido usando a tag INS (forma semântica) para dar ênfase e indicar que ele foi adicionado depois.

Existe também a tag U usada para sublinhar um texto de forma não semântica.

Texto Sobrescrito

Para inserir coisas do tipo x20+3 usamos a tag SUP.

Texto Subscrito

Para inserir coisas do tipo H2O usamos a tag SUB.
