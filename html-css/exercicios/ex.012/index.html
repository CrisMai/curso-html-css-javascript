Inserindo Vídeos Hospedados Localmente

Este vídeo está hospedado no meu próprio servidor.

Para inserir um vídeo é usado a tag VIDEO. É criado alguns sources e inserido os vários formatos de vídeos (mp4, webm, m4v, ogv) para assim, o vídeo abrir em diferentes navegadores.

Para inserir uma capa na apresentação de seu vídeo é usado POSTER.

Seu navegador não tem compatibilidade com reprodução de vídeos.

Inserindo Vídeos do Youtube

Para inserir um vídeo do Youtube basta escolher o vídeo que deseja, clicar em compartilhar, em seguida em incorporar e copiar o código em HTML do vídeo.

Em seguida, cole no seu documento HTML.
