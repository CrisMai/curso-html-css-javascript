Usando o Método Estilos Locais/Internos

Estilos Internos

Para aplicar os Estilos Locais/Internos na área HEAD cria uma tag STYLE, sendo que tudo o que for escrito dentro dessa tag será considerado configuração de estilo.

Capítulo 2

Esse método é utilizado quando tiver uma página simples e quiser fazer configurações locais que não serão aplicadas em outros documentos.

Capítulo 3

Esse tipo de declaração é aplicada na mesma página onde estão as tags a serem utilizadas, mas serão declaradas de forma global, evitando assim inserir o estilo em cada tag específica.
