Mídias em HTML5

Imagem Dinâmica

A tag PICTURE permite que sejam criados o SOURCE MEDIA, sendo que esses sources permitem adaptar o tamanho de suas imagens ao tamanho da tela do usuário.

Obs: a base é a imagem padrão que você quer carregar, accima dela você vai colocando os sources na ordem que desejar.

Tente abrir esse site em vários dispositivos diferentes ou simplesmente aumente ou diminua o tamanho de seu navegador.

Reproduzindo Aúdio

Vamos reproduzir um aúdio em html5.

A maneira mais simples de reproduzir um aúdio é através da tag AUDIO SRC usando também CONTROLS AUTOPLAY.

Alguns navegadores não têm compatibilidade com alguns formatos de arquivos de aúdio, por exemplo com o mp3. Nesses casos, temos outros formatos como o wav, o ogg.

Dessa forma, se você quiser ter uma maior compatibilidade com os navegadores é ncessário outra forma de adicionar o aúdio:

O navegador vai tentar reproduzir conforme a ordem acima...se ele não conseguir reproduzir pode colocar um link para baixar o arquivo.

Infelizmente seu navegador não consegue reproduzir aúdio. Clique aqui para baixar o arquivo MP3.
