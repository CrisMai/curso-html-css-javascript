Trabalhando com listas

Listas Ordenadas

Para criar uma lista ordenada, basta usar a tag OL. Cada item da lista deve ser colocado entre a tag LI.

Para personalizar uma lista é usado o parâmetro TYPE onde terá algumas opções disponíveis (1,A,a,I,i). O início de uma lista começa por padrão em 1; se caso quiser que a lista comece a partir do 4, por exemplo, deve usar o START.

  1. Acordar
  2. Escovar os dentes
  3. Tomar café
  4. Ligar para a Maria
  5. Ir para o trabalho
  6. Almoçar
  7. Ir no mercado
  8. Ir para faculdade
  9. Jantar
  10. Dormir

Listas Não Ordenadas

Para criar uma lista não ordenada seguimos o padrão acima, porém utilizamos a tag UL. Para personalizar usamos também o parâmetro TYPE (disc,circle,square).

  • Arroz
  • Tomate
  • Frutas
  • Leite
  • Macarrão

Lista de Definição

Para criar uma lista de definição usamos a tag DL, sendo que na lista de definição temos o termo (dt) e a descrição (dd).

Termo
Descrição
HTML
Linguagem de marcação de conteúdo de um site.
CSS
Linguagem de marcação para criação de design de um site.
JavaScript
Linguagem de programação para criação de interatividade de um site.
