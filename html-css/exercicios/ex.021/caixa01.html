Exemplo de Caixa Box-level

Parágrafos são exemplos de box-level e os links são exemplos de caixas inline-level. Vamos ver como isso funciona.

Para centralizar caixas é usado o MARGIN: AUTO. Para fazer um tracejado dentro da margenm é usado o OUTLINE.

Podemos simplificar as configurações usando o SHORTHAND, ou seja, diminuindo as linhas do código.
