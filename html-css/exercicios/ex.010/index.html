Trabalhando com Links

Usando Links Externos

Para criar um link externo será usado a tag A, sendo que terá outro parâmetro obrigatório o HREF (referência hipertexto).

Você pode acessar o meu repositório repositório público no GitHub.

Para abrir o link em uma nova aba é necessário usar o parâmetro "TARGET= _BLANK" e também o parâmetro REL="EXTERNAL".
