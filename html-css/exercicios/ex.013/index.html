Usando o Método Estilos Inline

Estilos Inline

É um método usado para configurações pontuais, ou seja, para uma ou outra configuração muito pequena.

Para construir o design de um site é necessário usar os outros métodos das CSS.

Capítulo 2

Nesse método Inline o estilo é feito basicamente na mesma linha que o CSS e todas as configurações também são feitas na mesma linha.

Capítulo 3

Existem diversas maneiras de se inserir códigos CSS em um site: Inline; Declarando na mesma página; Utilizando um arquivo externo.

Para utilizar o método Inline basta inserir o style dentro da própria tag html que ele receberá o efeito.
