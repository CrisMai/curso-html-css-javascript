Personalizando Links

Por padrão, os navegadores mostram links inéditos em azul e os visitados em violeta e os links parecem sublinhados.

Você pode alterar essas configurações usando alguns pseudo-elementos e pseudo-classes.

a {} muda a cor do link; a:visited {} muda a cor dos links visitados; a:hover {} muda a cor do link quando passar o mousse por cima; a::after {} vai adicionar algum conteúdo no final; a::before {} adiciona algum conteúdo no início do link.

# = id; . = class; : = pseudo-class; :: = pseudo-element; > = clildren.
